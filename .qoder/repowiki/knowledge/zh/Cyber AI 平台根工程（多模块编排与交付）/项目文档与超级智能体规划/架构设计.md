纯文档目录，按用途分为三层：
- `docs/DEPLOYMENT.md`：面向开发者的部署手册，覆盖 JDK/Maven/Node.js/Ollama/MySQL 环境准备、`mvn spring-boot:run` + `npm run dev` 本地启动流程、基于 `docker-compose --profile ollama` 的 Docker 生产部署、application.yml 配置项速查表、FAQ 排障与 curl 快速测试。
- `docs/superpowers/specs/2026-07-04-spring-ai-cyber-platform-design.md`：技术规格书，定义前后端分层架构（Controller/Service/Mapper/Entity）、数据库 schema（conversation/chat_message）、REST API 清单、Vue3+Element Plus 赛博朋克主题规范及组件树。
- `docs/superpowers/plans/2026-07-04-cyber-ai-platform.md`：以 `- [ ] Step N` 复选框驱动的超智能体可执行计划，将规格拆为“后端基础改造→数据库层→服务与Controller改造”三阶段任务，每个 Step 附带预期命令与 commit message，供 agentic worker 逐项推进。
依赖方向单向：plan 依据 spec 细化，spec 与 plan 均不反向引用代码仓库。
